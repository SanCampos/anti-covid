Super Admin

The super admin is created by the root user - the creator of the website. The super admin username and password are then given to the appropriate city hall official designated by the city or municipality mayor. It is important to note that there is only one super admin per city or municipality.
  • Create and manage Admins
  • Set city announcements
  • Delete routes
  • All powers of the admin (must create an admin user for self first)
  1. Ask the root user for the username and password of your city - ensure that you have the proper documents to prove that you are allowed to be the super admin of your city.
  2. Go to - mypasahero.com/manage. Use the username and password obtained in Step 1 to login. If this is your first time logging in, it is strongly advised to change your password.
  3. Now you are in the Announcements page. If this is your first login, there will be no announcements at the moment (See how to number 3).
  1. Upper right hand corner of the website, click the “Change Password” button
  2. Enter the password given by the root user (or your current password) on “old password.” Type your new password on “new password.” Then retype your new password again on “new password confirmation.”
  3. Click the “Save changes” button
  1. Click the “Announcements” button on the top part of the screen.
  2. Here, you can see the present announcements to the commuters if there is any. Click the “Change” button to edit the announcement
  3. There are two types of announcements. The Major Announcement and the Minor Announcements
  4. To set an announcement, major or minor, type your announcement inside the big rectangular box. If you want this announcement to be displayed, click the appropriate checkbox to display it.
  5. Scroll down and click the “Save” button
  1. Click “Add Admin” button
  2. Create a username and password for the new admin. Note: usernames cannot be changed but passwords can be changed by the admin later on once the admin has logged in to his account.
  3. Fill in the rest of the form with the admin’s credentials.
  4. Click “Create Admin User” to create the admin
    5. *Note: Once an Admin has been created, he can never be deleted. He can only be deactivated and, if necessary, reactivated. The purpose of this is for easy auditing. Once an Admin has been created, he is automatically set to active status.
  1. Click the “All Admins” button to display all admins
  2. Look at the “Active Admins” table and click the minus sign “-” to deactivate the Admin
  3. Now the Admin will be under the “Inactive Admins” table where they cannot sign in to the Admin page anymore
  1. Click the “All Admins” button to display all admins
  2. Look at the “Inactive Admins” table and click the plus sign “+” to reactivate the Admin
  3. Now the Admin will be under the “Active Admins” table where they can sign in to the Admin page again
  1. Click the “All Routes” button to display all routes
  2. Click the “Delete” button for the route you want to delete
    3. WARNING: It is not recommended to delete a route as it will delete all the commuter applications and other data associated with it. It is better to simply deactivate the route from the Admin page (See Admin User guidelines).
